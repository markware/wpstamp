== Description ==

A custom product tool for WooCommerce and WordPress using FabricJS.
This is an old version not maintained, feel free to fork it or whatever.
It should still work fine in Woocommerce latest version!
== Installation ==

Install Grunt, and run the Grunt file. Activate it as you would normally do in a WordPress plugin
